本實驗的主要目標是利用單應性矩陣來補償USB相機拍攝影像中的
幾何變化。這些變化可能由視角變化、相機位置變動或鏡頭畸變所
引起，影響影像的幾何精度。透過單應性矩陣，我們能將一幅影像
中的點映射到另一幅影像中對應的位置，從而達到精確的影像對
齊。

1.設置固定相機鏡頭：確保相機鏡頭穩定並固定在指定位
置。 
2.擺放標記物：至少擺放四個標記物在相機視野內，並確保
它們的位置可以被清楚識別。 
3.記錄標記物的實際距離：測量並記錄每個標記物與相機鏡
頭X軸及Y軸的真實距離。 
4.拍攝照片：使用固定相機拍攝一張包含所有標記物的照
片。 
5. 開啟照片並記錄座標：將拍攝的照片在小畫家中開啟，
並記錄每個標記物在照片中的座標，座標以小畫家左下角
為基準。
